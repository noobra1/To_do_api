# To-Do API Manager

This Python project demonstrates CRUD operations (GET, POST, PUT, DELETE) using the [JSONPlaceholder API](https://jsonplaceholder.typicode.com/).

## Features
- Fetch first 5 To-Do tasks
- Add a new task
- Update an existing task
- Delete a task

## Requirements
- Python 3.x
- `requests` library

## Usage
```bash
python todo_manager.py
